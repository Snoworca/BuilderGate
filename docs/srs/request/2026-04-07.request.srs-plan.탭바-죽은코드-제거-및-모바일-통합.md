# 요구사항 정리 문서

## 사용자 원문
> 데스크탑 기반으로 바꾸고 죽은 코드를 제거하도록 합시다.

## 배경
- 데스크탑 모드의 탭 상단에 컬러 바가 있지만 모바일 모드에서는 보이지 않음
- `TabBar` 컴포넌트(`components/TabBar/`)가 존재하나 어디에서도 사용되지 않는 죽은 코드
- 실제로는 `WorkspaceTabBar`(`components/Workspace/`)만 데스크탑+모바일 모두에서 사용 중

## 핵심 목표
1. 죽은 코드(`TabBar` 컴포넌트 및 관련 전용 파일) 제거
2. 모바일에서도 데스크탑과 동일하게 탭 상단 컬러 바가 보이도록 수정

## 제약사항
- `useTabManager.ts`의 `PendingOp` 타입은 `MdirPanel.tsx`에서 사용하므로 삭제 불가
- `WorkspaceTabBar`가 유일한 탭 바이므로 이를 기준으로 통합
