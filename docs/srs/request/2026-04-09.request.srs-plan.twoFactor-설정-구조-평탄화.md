# 요구사항 정리: twoFactor 설정 구조 평탄화

## 원문
config.json5의 twoFactor.totp 중첩 구조를 평탄화한다.

변경 전:
```json5
twoFactor: {
  externalOnly: false,
  totp: {
    enabled: true,
    issuer: "BuilderGate",
    accountName: "admin"
  },
}
```

변경 후:
```json5
twoFactor: {
  enabled: true,
  externalOnly: false,
  issuer: "BuilderGate",
  accountName: "admin"
}
```

## 목표
- `twoFactor.totp` 중첩 객체를 제거하고 필드를 `twoFactor` 직하로 이동
- config.json5, config.json5.example, 관련 타입/스키마/서비스/테스트 코드를 모두 동기화

## 변경 유형
refactor (기능 변경 없음, 설정 구조 단순화)

- TOTP 기능 동작은 변경 없음
- TypeScript 빌드 오류 없어야 함
