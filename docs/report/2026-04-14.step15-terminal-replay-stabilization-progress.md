## Important Fixes

### Frontend

- `TerminalView` now deduplicates geometry emission and re-enables stdin on focus/click, not only on fit.
- `TerminalContainer` skips re-applying an identical `screen-snapshot` payload and immediately ACKs it instead.
- Tab-mode metadata uses a fixed slot height so active/inactive workspace transitions do not change terminal height purely because the metadata row appears or disappears.

### Backend

- `SessionManager.resize()` records `resize_requested`, returns early on unchanged geometry, and emits `resize_skipped` instead of mutating PTY or replay lineage.
- `SessionManager.getScreenSnapshot()` waits for the relevant headless write boundary before serializing authoritative state.
- `WsRouter.refreshReplaySnapshots()` suppresses stale concurrent refresh completions for the same session.
- `WsRouter.handleSubscribe()` now survives disconnect during async snapshot loading.

## Residual Risk

- The automated suite now proves authority-path recovery and resize dedupe, but it still does not fully automate the real Codex `1..500` vertical-output bounce workload.
- Manual validation on `https://localhost:2002` is still required before calling the Codex TUI duplicate-output issue fully closed.
