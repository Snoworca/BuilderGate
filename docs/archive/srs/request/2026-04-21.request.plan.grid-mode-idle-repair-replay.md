# 요청 정리: Grid Mode Idle Repair Replay

작성일: 2026-04-21

## 배경

- Grid mode에서 `codex`, `hermes` 같은 TUI를 띄우면 줄바꿈과 라인 배치가 종종 깨진다.
- 사용자가 그리드 셀 크기를 살짝 조절하면 다시 렌더링되며 정상화된다.
- 이 수동 복구 과정을 idle 진입 시 자동으로 재현할 수 있는지 검토가 필요하다.

## 요구사항

1. Grid mode에서 깨진 TUI 화면을 idle 진입 시 자동으로 한 번 복구하는 방향으로 구현 계획을 수립한다.
2. 단순 React rerender가 아니라, 실제 수동 크기 조절이 유발하는 복구 과정과 동등한 효과를 목표로 한다.
3. geometry를 실제로 바꾸는 synthetic resize보다, 가능하면 geometry mutation 없는 복구 경로를 우선 검토한다.
4. tab mode와 일반 non-TUI 세션의 기존 동작은 건드리지 않는 방향을 우선한다.
5. 구현 전에 검증 가능한 계획 문서로 정리하고, 이후 plan-driven-coder가 바로 실행할 수 있어야 한다.

## 초기 방향

- 1순위: `idle + quiet window` 뒤에 authoritative snapshot 기반 repair replay를 1회 수행
- 2순위: 필요 시 `fit()` 1회 보조 실행
- 제외: 1차 구현에서 synthetic resize를 기본 복구 방식으로 채택하지 않음
