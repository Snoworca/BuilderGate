# BuilderGate 프로젝트 구조 (2026-04-02)

코딩 에이전트 병렬 운용을 위한 웹 기반 통합 개발 환경의 전체 구조를 기록한다.

## 프로젝트 개요

| 항목 | 값 |
|------|-----|
| 프로젝트명 | BuilderGate |
| 목적 | 브라우저 기반 다중 셸 세션 관리 + 파일 탐색/편집 + 에이전트 오케스트레이션 |
| 서버 포트 | 4242 (HTTPS) |
| 프론트엔드 포트 | 4545 (Vite dev) |
| 통신 방식 | WebSocket 단일 채널 (`/ws`) |

## 기술 스택 요약

- **Backend**: Node.js + Express + TypeScript, node-pty, JWT, WebSocket
- **Frontend**: React 19 + TypeScript, Vite 7, xterm.js 6
- **Communication**: WebSocket (Step 8에서 SSE+HTTP → WS 마이그레이션 완료)
- **Config**: JSON5 + Zod validation + AES-256-GCM 암호화
- **Security**: HTTPS(자동 생성 인증서) + JWT HS256 + 선택적 2FA(이메일 OTP)
