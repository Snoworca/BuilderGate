# 03. 프론트엔드 아키텍처

## 디렉토리 구조

```
frontend/
├── package.json          # React 19, xterm.js 6, Vite 7
├── vite.config.ts        # 포트 4545, HMR via WSS(4242)
├── tsconfig.json         # tsconfig.app.json + tsconfig.node.json 참조
├── eslint.config.js      # ESLint 9 flat config
├── playwright.config.ts  # E2E 테스트
├── index.html            # Vite 엔트리 (#root)
└── src/
    ├── main.tsx          # React 루트 (<AuthProvider> → <App>)
    ├── App.tsx           # 앱 셸 (워크스페이스 기반)
    ├── index.css         # 글로벌 리셋
    ├── contexts/         # React 컨텍스트
    ├── hooks/            # 커스텀 훅
    ├── components/       # UI 컴포넌트
    ├── services/         # API 클라이언트
    ├── types/            # TypeScript 타입
    ├── utils/            # 유틸리티
    └── styles/           # 글로벌 CSS 변수
```

## 앱 구조 (`App.tsx`)

```
<AuthGuard>
  <WebSocketProvider>
    <AppContent>
      ├── <aside> WorkspaceSidebar (데스크톱) / MobileDrawer (모바일)
      └── <main>
          ├── <Header> (CWD, 뷰 모드 토글, 설정, 로그아웃)
          ├── Tab 모드: <TerminalContainer> + <MetadataRow>
          ├── Grid 모드: <GridContainer> (√n 기반 그리드)
          └── Settings 화면: <SettingsPage>
    </AppContent>
  </WebSocketProvider>
</AuthGuard>
```

## 컨텍스트 (`src/contexts/`)

| 파일 | 역할 |
|------|------|
| `AuthContext.tsx` | 전역 인증 상태 관리. `login`, `verify2FA`, `logout`, `refreshToken`. localStorage 토큰 복원 |
| `WebSocketContext.tsx` | 단일 WSS 연결(`/ws?token=...`). 지수 백오프 재연결(최대 10회, 1초→30초). 세션별 핸들러 구독, 워크스페이스 이벤트 핸들러 |

## 훅 (`src/hooks/`)

| 파일 | 역할 |
|------|------|
| `useWorkspaceManager.ts` | **핵심 상태 관리자**. 워크스페이스/탭/그리드 CRUD. `/api/workspaces`에서 초기 상태 fetch. WS 이벤트로 실시간 동기화. localStorage에 활성 워크스페이스 ID 저장 |
| `useSession.ts` | 레거시 세션 목록 관리 (파일 매니저용) |
| `useTabManager.ts` | 레거시 탭 매니저. 터미널+파일 탭 통합, pendingOp(파일 복사/이동), 뷰어 상태 |
| `useCwd.ts` | 세션별 CWD 조회. 새 세션 시 1회 fetch, 활성 세션 5초 폴링 |
| `useHeartbeat.ts` | JWT 갱신(15분 주기). 최대 2회 재시도, 실패 시 로그아웃 |
| `useResponsive.ts` | 모바일 감지(768px), 사이드바 열림/닫힘 |
| `useDragReorder.ts` | 드래그 앤 드롭 순서 변경 (터치 300ms 롱프레스 지원) |
| `useContextMenu.ts` | 우클릭 컨텍스트 메뉴 위치/표시 상태 |
| `useKeyboardNav.ts` | 키보드 네비게이션 (방향키, Enter, Backspace) |
| `useFileBrowser.ts` | 파일 브라우저 상태: 경로, 디렉토리 목록, 선택, 히스토리 |
| `useFileContent.ts` | 파일 내용 fetch + 캐시 |
| `usePinchZoom.ts` | 터치 핀치 줌 (터미널 폰트 크기) |

## 컴포넌트 (`src/components/`)

### Terminal/

| 파일 | 역할 |
|------|------|
| `TerminalContainer.tsx` | WS 세션 이벤트 구독, output→TerminalView.write(), status/cwd→부모 콜백. Memo 최적화 |
| `TerminalView.tsx` | xterm.js Terminal 인스턴스 + addon-fit. Imperative handle(`write`, `resize`). WS로 input/resize 전송. 핀치 줌 |
| `FontSizeToast.tsx` | 핀치 줌 시 폰트 크기 토스트 |

### Workspace/

| 파일 | 역할 |
|------|------|
| `WorkspaceSidebar.tsx` | 좌측 사이드바. 워크스페이스 목록, 드래그 순서 변경, 활성 표시, 실행 중 터미널 수 배지 |
| `WorkspaceItem.tsx` | 워크스페이스 행: 이름, 탭 수, 컨텍스트 메뉴 |
| `WorkspaceTabBar.tsx` | 수평 탭 바. 색상 코딩, 드래그 순서 변경, 더블클릭 이름 변경 |
| `MobileDrawer.tsx` | 모바일용 슬라이드인 사이드바 |
| `EmptyState.tsx` | 빈 탭 안내 + "Add Terminal" 버튼 |
| `DisconnectedOverlay.tsx` | 세션 연결 끊김 오버레이 + "Restart" 버튼 |

### Grid/

| 파일 | 역할 |
|------|------|
| `GridContainer.tsx` | √n 기반 자동 그리드 레이아웃. tabOrder 또는 sortOrder로 정렬 |
| `GridCell.tsx` | 그리드 셀: 터미널 또는 DisconnectedOverlay, 탭 색상 테두리, 메타데이터 |
| `EmptyCell.tsx` | 빈 셀 "+" 플레이스홀더 |

### FileManager/

| 파일 | 역할 |
|------|------|
| `MdirPanel.tsx` | Mdir 스타일 파일 매니저 메인 패널 |
| `MdirFileList.tsx` | 스크롤 가능 파일/디렉토리 목록 (아이콘, 크기, 날짜) |
| `MdirHeader.tsx` | 현재 경로 표시 |
| `MdirFooter.tsx` | 기능키 힌트 (F5 Copy, F6 Move, F7 Mkdir, F8 Delete) |
| `FileOperationDialog.tsx` | 파일 작업 확인 모달 |

### Auth/

| 파일 | 역할 |
|------|------|
| `AuthGuard.tsx` | 인증 가드: LoginForm 또는 TwoFactorForm 렌더링 |
| `LoginForm.tsx` | 비밀번호 입력 폼 |
| `TwoFactorForm.tsx` | 6자리 OTP 입력 폼 |

### 기타 컴포넌트

| 디렉토리 | 역할 |
|-----------|------|
| `Header/` | 상단 바: 앱 이름, CWD, 뷰 모드 토글, 설정, 로그아웃 |
| `MetadataBar/` | 탭 하단 메타데이터: 탭 이름, 셸 타입, 상태, CWD |
| `Modal/` | ConfirmModal, RenameModal, AddTabModal, ShellSelectModal, TabContextModal |
| `ContextMenu/` | 우클릭 플로팅 메뉴 |
| `Settings/` | 설정 페이지: 비밀번호, 2FA, CORS, PTY, 세션, 파일 매니저 설정 |
| `StatusBar/` | 글로벌 상태 바 (연결 상태, 세션 수) |
| `Sidebar/` | 레거시 사이드바 (워크스페이스 전환 전) |
| `TabBar/` | 레거시 탭 바 |

## 서비스 (`src/services/`)

| 파일 | 역할 |
|------|------|
| `api.ts` | 전체 HTTP API 호출. authApi, sessionApi, fileApi, workspaceApi, settingsApi 네임스페이스. Bearer 토큰 + x-client-id 헤더 자동 주입 |
| `tokenStorage.ts` | localStorage 기반 JWT 저장: setToken, getToken, removeToken, isExpired |

## 타입 (`src/types/`)

| 파일 | 정의 내용 |
|------|-----------|
| `index.ts` | Session, SessionStatus, ShellType, LoginResponse, AuthState, DirectoryEntry 등 |
| `workspace.ts` | Workspace, WorkspaceTab, WorkspaceTabRuntime, GridLayout, TAB_COLORS(8색) |
| `ws-protocol.ts` | ClientWsMessage, ServerWsMessage (서버 미러) |
| `settings.ts` | EditableSettingsKey, SettingsSnapshot, SettingsPatchRequest 등 |

## 유틸리티 (`src/utils/`)

| 파일 | 역할 |
|------|------|
| `pathUtils.ts` | 크로스 플랫폼 경로 표시 (getPathSeparator, joinPath, truncatePathLeft) |
| `viewableExtensions.ts` | 뷰어 지원 확장자 Set (md/js/ts/py/go 등) |
