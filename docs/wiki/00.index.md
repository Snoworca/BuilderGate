# BuilderGate 위키

이 디렉터리는 `C:\Work\git\_Snoworca\ProjectMaster` 저장소를 2026-04-16 기준으로 다시 분석해 정리한 운영용 위키입니다.

이 위키의 목적은 "새 코딩 에이전트가 이 저장소에 들어왔을 때 현재 활성 구조를 빠르게 파악하고, 잘못된 문서나 레거시 코드에 속지 않도록" 만드는 것입니다.

## 소스 우선순위

문서가 충돌하면 아래 순서를 따릅니다.

1. 현재 코드
2. `docs/wiki`
3. 최신 검증/조사 문서
4. 오래된 계획 문서와 SRS
5. `README.md`, `CLAUDE.md`

특히 `README.md`와 `CLAUDE.md`에는 과거 포트, 과거 통신 방식, 과거 UI 구조가 남아 있으므로 현재 상태의 단일 진실 원본으로 쓰면 안 됩니다.

## 빠른 사실만 먼저

- 제품 성격: HTTPS 기반 브라우저 터미널/워크스페이스 관리 도구
- 현재 기본 개발 진입점: 루트 `node dev.js`
- 현재 기본 포트: `http://localhost:2001` 리다이렉트, `https://localhost:2002` 앱, `http://localhost:2003` Vite
- 핵심 통신: REST + 단일 WebSocket `/ws`
- 인증: 비밀번호 + 선택적 TOTP
- 현재 핵심 구조: 서버 headless snapshot 권위 + 프런트 terminal runtime 재사용
- 활성 UI: 워크스페이스, 탭, 그리드, 설정
- 비활성/레거시: 파일 매니저 메인 UI 연결, 구형 pane/session 스택, 일부 오래된 Playwright spec
