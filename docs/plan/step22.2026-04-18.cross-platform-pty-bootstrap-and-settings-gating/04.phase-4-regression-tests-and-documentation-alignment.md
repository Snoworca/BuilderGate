## 테스트 시나리오

### 정상 케이스

- Given: clean install 환경이다
  When: 각 OS bootstrap 테스트를 수행한다
  Then: 생성 config와 startup contract가 일치한다

- Given: non-Windows stale config fixture가 있다
  When: loader 테스트를 수행한다
  Then: startup-safe normalized config가 나온다

### 예외 케이스

- Given: hidden PTY field가 있는 settings snapshot이다
  When: unrelated save 검증을 수행한다
  Then: validation mismatch로 실패하지 않는다

### 경계값 테스트

- Given: legacy `maxBufferSize`와 Windows 전용 PTY 값이 섞여 있는 fixture다
  When: normalization과 persistence 관련 테스트를 수행한다
  Then: legacy compatibility와 cross-platform normalization이 함께 유지된다
