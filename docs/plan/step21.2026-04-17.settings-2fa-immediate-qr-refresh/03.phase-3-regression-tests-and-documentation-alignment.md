## 테스트 시나리오

### 정상 케이스

- Given: 2FA가 꺼져 있다
  When: 설정 페이지에서 켜고 저장한다
  Then: 같은 화면에서 QR 이미지가 나타난다

- Given: 2FA가 켜져 있다
  When: accountName을 바꾸고 저장한다
  Then: URI 텍스트가 새 accountName을 포함한다

### 예외 케이스

- Given: TOTP secret 손상 상태다
  When: 설정 페이지에서 enable 또는 metadata save를 수행한다
  Then: save 응답 warning과 `registered=false` 상태가 노출되고 QR image는 비어 있다

### 경계값 테스트

- Given: 저장 버튼 연속 클릭이 발생한다
  When: 첫 번째 저장이 끝나기 전에 두 번째 클릭이 들어온다
  Then: 중복 refresh race 없이 마지막 저장 결과만 남는다
