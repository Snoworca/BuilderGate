# Phase 1: Vendor Drag-Start Mutation Removal

## Objective

drag start 시점의 `hide()` 호출과 그에 의존한 tree mutation을 제거한다.

## Target Files

- `frontend/node_modules/react-mosaic-component/lib/MosaicWindow.js`
- `frontend/node_modules/react-mosaic-component/lib/util/mosaicUpdates.js`
- `frontend/patches/react-mosaic-component+6.1.1.patch`

## Planned Changes

1. drag start에서 `mosaicActions.hide(props.path)` 제거
2. source node는 live tree에서 유지
3. preview/drag source 초기화가 tree mutation 없이 유지되는지 확인

## Test

- drag start 직후 source tile rect stability 확인
- drag preview/guide가 계속 뜨는지 확인
