# Step 6 구현 계획: tmux-Style Pane Split System

**Version**: 1.0.0
**Date**: 2026-03-21
**SRS Reference**: `docs/archive/spec/srs.step6.md`
**Research Reference**: `docs/research/tmux-feature-research.md`

---

## 프로젝트 개요

BuilderGate에 tmux 스타일 화면 분할(Pane Split) 시스템을 추가한다. 재귀적 바이너리 트리 기반 수평/수직 분할, 컨텍스트 메뉴 UX, 모바일 캐러셀, IndexedDB 영속화, 키보드 단축키를 구현한다.

**서버 변경 없음** — 기존 세션 생성/삭제/SSE API를 그대로 활용한다.
