## 3. 클래스/인터페이스 다이어그램

### 3.1 핵심 데이터 모델

```typescript
// types/pane.types.ts

type PaneNode = PaneLeaf | PaneSplit;

interface PaneLeaf {
  type: 'terminal';
  id: string;           // UUID
  sessionId: string;    // PTY 세션 ID
}

interface PaneSplit {
  type: 'split';
  id: string;           // UUID
  direction: 'horizontal' | 'vertical';
  ratio: number;        // 0.15 ~ 0.85
  children: [PaneNode, PaneNode];
}

interface PaneLayout {
  root: PaneNode;
  focusedPaneId: string;
  zoomedPaneId: string | null;
}

type PresetType =
  | 'single'
  | 'vertical-2'
  | 'horizontal-2'
  | 'quad'
  | 'main-side'
  | 'agent-monitor';

type Direction = 'horizontal' | 'vertical';
type FocusDirection = 'up' | 'down' | 'left' | 'right';
```

### 3.2 IndexedDB 스키마 모델

```typescript
// utils/paneDb.ts

interface PaneLayoutRecord {
  sessionId: string;     // keyPath
  layout: PaneLayout;
  updatedAt: number;
}

interface SavedLayoutRecord {
  id: string;            // keyPath, UUID
  name: string;
  layout: PaneLayout;    // sessionId = '__placeholder__'
  isBuiltIn: boolean;
  paneCount: number;
  createdAt: number;
}

interface SessionMetaRecord {
  sessionId: string;     // keyPath
  groupId?: string;
  color?: string;
  lastConnected: number;
}
```

### 3.3 훅 의존성 관계

```
App.tsx
  ├─ useSession()         → sessions[], activeSessionId
  ├─ useTabManager()      → tabs[], activeTabId
  └─ usePaneManager(      → layout, prefixMode, splitPane(), ...
       activeTabId,
       usePaneDB()        → saveLayout(), loadLayout(), ...
     )
       └─ paneTree.ts     → splitPane(), closePane(), ... (순수함수)
```

## 7. 의존성 방향 (레이어 다이어그램)

```
┌──────────────────────────────────────┐
│ Layer 4: Components (React)          │
│  SplitPane, PaneCarousel,            │
│  PaneRenderer, PaneResizer,          │
│  PaneNumberOverlay                   │
├──────────────────────────────────────┤
│ Layer 3: Hooks (React State)         │
│  usePaneManager, usePaneDB           │
├──────────────────────────────────────┤
│ Layer 2: Utilities (Pure Functions)  │
│  paneTree.ts (14 functions)          │
├──────────────────────────────────────┤
│ Layer 1: Data (Browser Storage)      │
│  paneDb.ts (IndexedDB schema)        │
│  pane.types.ts (TypeScript types)    │
└──────────────────────────────────────┘

의존성 방향: Layer 4 → 3 → 2 → 1 (상위가 하위에만 의존)
```

순환 의존성을 엄격히 금지한다. `paneTree.ts`는 React에 의존하지 않는 순수 함수 모듈이다.
