#### 2.1.3 `splitPane(paneId, direction)` -- async, no optimistic update

```typescript
async splitPane(paneId: string, direction: Direction): Promise<void>
```

1. **Guard**: if `!canSplit`, return early (paneCount >= 8 or treeDepth >= 4).
2. **Guard**: if `layout.zoomedPaneId !== null`, return early (no split during zoom).
3. **Find source pane**: `findPane(layout.root, paneId)` to get sessionId, then resolve parent CWD from CWD map.
4. **Create PTY session**: `POST /api/sessions` via `createSession('Sub-Terminal', undefined, parentCwd, false)`.
   - On failure: show error in StatusBar for 3 seconds. Do NOT modify tree. Return early.
   - **No optimistic update**: tree is only modified after API call succeeds.
5. **Update tree**: `paneTreeSplit(layout.root, paneId, direction, newSession.id)` to get new root.
6. **Update layout state**: new root, `focusedPaneId` = new PaneLeaf (second child of new split).
7. **Trigger debounced save** (300ms).

```typescript
async splitPane(paneId: string, direction: Direction): Promise<void> {
  if (!canSplit || layout.zoomedPaneId) return;

  // 1. API call first -- no optimistic update
  let newSession: Session;
  try {
    newSession = await createSession('Sub-Terminal', undefined, parentCwd, false);
    if (!newSession) throw new Error('Session creation returned null');
  } catch (err) {
    showStatusBarError('Failed to create session for split');
    return; // tree was never modified
  }

  // 2. Only now update the tree
  const newRoot = paneTreeSplit(layout.root, paneId, direction, newSession.id);
  const newLeaves = flattenPaneTree(newRoot);
  const newPane = newLeaves.find(l => l.sessionId === newSession.id);

  setLayout({
    root: newRoot,
    focusedPaneId: newPane?.id ?? layout.focusedPaneId,
    zoomedPaneId: null,
  });

  debouncedSave();
}
```

#### 2.1.4 `closePane(paneId)` -- async

```typescript
async closePane(paneId: string): Promise<void>
```

1. **Guard**: if `paneCount <= 1`, return early (cannot close last pane).
2. **Guard**: if `layout.zoomedPaneId !== null`, return early (no close during zoom).
3. **Find pane**: `findPane(layout.root, paneId)` to get sessionId.
4. **Update tree**: `paneTreeClose(layout.root, paneId)`. SSE closes automatically via TerminalContainer unmount.
5. **Delete PTY session**: `deleteSession(sessionId)`. Fire-and-forget with error logging.
6. **Update focus**: if closed pane was `focusedPaneId`, set focus to first leaf of new tree.
7. **Clear zoom**: if closed pane was `zoomedPaneId`, set `zoomedPaneId = null`.
8. **Trigger debounced save** (300ms).

```typescript
async closePane(paneId: string): Promise<void> {
  if (paneCount <= 1 || layout.zoomedPaneId) return;

  const leaf = findPane(layout.root, paneId);
  if (!leaf) return;

  const newRoot = paneTreeClose(layout.root, paneId);
  if (!newRoot) return; // guard: paneCount > 1 prevents this

  const leaves = flattenPaneTree(newRoot);
  const newFocus = leaves[0]?.id ?? layout.focusedPaneId;

  setLayout({
    root: newRoot,
    focusedPaneId: paneId === layout.focusedPaneId ? newFocus : layout.focusedPaneId,
    zoomedPaneId: null,
  });

  // Fire-and-forget session deletion
  deleteSession(leaf.sessionId).catch(console.error);

  debouncedSave();
}
```

#### 2.1.5 `closeOtherPanes(keepPaneId)` -- async

```typescript
async closeOtherPanes(keepPaneId: string): Promise<void>
```

1. **Flatten tree**: `flattenPaneTree(layout.root)`, filter out `keepPaneId`.
2. **Delete sessions**: for each pane to close, `deleteSession(pane.sessionId)`. Use sequential iteration; log failures, continue.
3. **Report partial failures**: if any deletions failed, show count in StatusBar.
4. **Build new layout**: single PaneLeaf with kept pane's data.
5. **Update state**: new layout, `focusedPaneId = keepPaneId`, `zoomedPaneId = null`.
6. **Trigger immediate save** (not debounced).

#### 2.1.6 Focus Management

**`setFocusedPane(paneId)`**:
- Validate paneId exists via `findPane`.
- If `swapSource !== null`: call `executeSwap(paneId)` instead of simple focus change.
- Update `layout.focusedPaneId`.
- Trigger debounced save (300ms).

**`moveFocus(direction: FocusDirection)`**:
- **DOM-based algorithm** (SRS section 3.5):
  1. Query all `[data-pane-id]` elements via `document.querySelectorAll`.
  2. For each, call `getBoundingClientRect()` to get center coordinates.
  3. From current focused pane's center, cast a ray in the requested direction.
  4. Find closest PaneLeaf whose bounding box intersects the ray.
  5. If found, `setFocusedPane(result.id)`. If not, no-op.
- **Fallback**: if DOM elements unavailable (e.g., during SSR or initialization), use `paneTree.getAdjacentPane()` tree-structural algorithm.

**`cycleFocus()`**:
- `flattenPaneTree(layout.root)` to get ordered leaf list.
- Find current focused index.
- Set focus to `(index + 1) % leaves.length`.

#### 2.1.7 Zoom

**`toggleZoom(paneId?: string)`**:
- If `paneCount === 1`: no-op (no point zooming single pane).
- If `layout.zoomedPaneId` is set:
  - If same pane or no paneId: unzoom (`zoomedPaneId = null`).
- If `layout.zoomedPaneId` is null:
  - Set `zoomedPaneId = paneId ?? layout.focusedPaneId`.
- Trigger debounced save (300ms).
- SplitPane/PaneRenderer handles display:none for non-zoomed panes.

#### 2.1.8 Resize

**`resizePane(splitId, ratio)`**:
- `paneTreeResize(layout.root, splitId, ratio)` (clamping 0.15-0.85 inside pure function).
- Update layout state immediately (**no debounce during drag** for smooth visual feedback).
- Mark `dirtyRef.current = true`.
- Save is triggered only by caller on `pointerup` via `immediateSave()`.

**`equalizePanes(splitId)`**:
- `equalizeRatios(layout.root, splitId)`.
- Update state. Debounced save (300ms).

**`toggleDirection(splitId)`**:
- `paneTreeToggleDir(layout.root, splitId)`.
- Update state. Debounced save (300ms).

#### 2.1.9 Swap

**`startSwap(paneId)`**:
- Set `swapSource = paneId`.
- UI shows dashed border on source pane.

**`executeSwap(targetPaneId)`**:
- Guard: if `swapSource === null` or `swapSource === targetPaneId`, cancel and return.
- `swapPanes(layout.root, swapSource, targetPaneId)`.
- Update state. Set `swapSource = null`. Debounced save.

**`cancelSwap()`**:
- Set `swapSource = null`.

#### 2.1.10 Preset / Layout (async)

**`applyPreset(preset: PresetType)`**:
1. Determine required pane count from `BUILT_IN_PRESETS`.
2. Close existing sessions: flatten tree, `deleteSession(sessionId)` for each. Fire-and-forget, continue on failure.
3. Create new sessions: `createSession()` for each required pane.
   - If any creation fails: delete already-created sessions (rollback), restore original layout, show error.
4. Build layout: `buildPresetLayout(preset, sessionIds)`.
5. Update state. Immediate save.

**`saveLayout(name: string)`**:
- Delegate to `paneDB.savePreset(name, layout)`.
- Return generated ID.

**`loadLayout(layoutId: string)`**:
1. Load saved layout record via `paneDB.loadPresets()`, find by ID.
2. Close all current sessions (same as applyPreset step 2).
3. Create required number of new sessions.
4. Replace `PLACEHOLDER_SESSION_ID` values in loaded layout with new session IDs.
5. Update state. Immediate save.

#### 2.1.14 Initialization

On mount (and when `activeTabId` / `activeSessionId` changes):

1. **Load from IndexedDB**: `paneDB.loadLayout(activeSessionId)`.
2. **If layout found**: reconcile with server sessions:
   - Walk loaded PaneNode tree. For each PaneLeaf, check sessionId exists in server session list.
   - Remove PaneLeaf nodes whose sessionId is missing (using `paneTreeClose` iteratively).
   - If all leaves removed, create default single-pane layout with `activeSessionId`.
   - If saved `focusedPaneId` was removed, set to first valid leaf.
3. **If no layout found**: `createDefaultLayout(activeSessionId)`.
4. Set `initialized = true`.

Reset `initialized` when `activeTabId` changes to trigger fresh load.

#### 2.1.15 Save Timing

| Event | Save Type | Debounce |
|-------|-----------|----------|
| splitPane (after API success) | debounced | 300ms |
| closePane | debounced | 300ms |
| setFocusedPane | debounced | 300ms |
| toggleZoom | debounced | 300ms |
| resizePane (during drag) | dirty flag only | - |
| resizePane end (pointerup) | immediate | 0ms |
| Tab switch (activeTabId change) | immediate | 0ms |
| `visibilitychange` → hidden | immediate | 0ms |
| `beforeunload` | best-effort immediate | 0ms |

```typescript
const debouncedSave = useCallback(() => {
  if (saveTimerRef.current) clearTimeout(saveTimerRef.current);
  dirtyRef.current = true;
  saveTimerRef.current = setTimeout(() => {
    if (activeSessionId) {
      paneDB.saveLayout(activeSessionId, layout);
      dirtyRef.current = false;
    }
  }, 300);
}, [activeSessionId, layout, paneDB]);

const immediateSave = useCallback(() => {
  if (saveTimerRef.current) clearTimeout(saveTimerRef.current);
  if (activeSessionId && dirtyRef.current) {
    paneDB.saveLayout(activeSessionId, layout);
    dirtyRef.current = false;
  }
}, [activeSessionId, layout, paneDB]);
```

#### 2.1.16 Event Listeners

Register `visibilitychange` and `beforeunload` in `useEffect`:

```typescript
useEffect(() => {
  const handleVisibilityChange = () => {
    if (document.hidden) immediateSave();
  };
  const handleBeforeUnload = () => {
    immediateSave(); // best-effort: may not complete before page unloads
  };
  document.addEventListener('visibilitychange', handleVisibilityChange);
  window.addEventListener('beforeunload', handleBeforeUnload);
  return () => {
    document.removeEventListener('visibilitychange', handleVisibilityChange);
    window.removeEventListener('beforeunload', handleBeforeUnload);
  };
}, [immediateSave]);
```

- **`vertical`** split (left/right): `flex-direction: row`.
- First child: `flex: {ratio} 1 0%`. Second child: `flex: {1 - ratio} 1 0%`.
- `PaneResizer` is placed between the two children.

#### 2.3.3 Drag Implementation (Pointer Events)

```typescript
function handlePointerDown(e: React.PointerEvent) {
  e.preventDefault();
  e.currentTarget.setPointerCapture(e.pointerId);
  onResizeStart();
  setDragging(true);
}

function handlePointerMove(e: React.PointerEvent) {
  if (!dragging) return;
  const parentRect = parentRef.current?.getBoundingClientRect();
  if (!parentRect) return;

  let newRatio: number;
  if (direction === 'vertical') {
    newRatio = (e.clientX - parentRect.left) / parentRect.width;
  } else {
    newRatio = (e.clientY - parentRect.top) / parentRect.height;
  }

  // Clamp to [MIN_RATIO, MAX_RATIO]
  newRatio = Math.max(PANE_CONSTANTS.MIN_RATIO, Math.min(PANE_CONSTANTS.MAX_RATIO, newRatio));

  // Additional clamp: minimum pane pixel size
  if (direction === 'vertical') {
    const minRatio = PANE_CONSTANTS.MIN_PANE_WIDTH / parentRect.width;
    const maxRatio = 1 - minRatio;
    newRatio = Math.max(minRatio, Math.min(maxRatio, newRatio));
  } else {
    const minRatio = PANE_CONSTANTS.MIN_PANE_HEIGHT / parentRect.height;
    const maxRatio = 1 - minRatio;
    newRatio = Math.max(minRatio, Math.min(maxRatio, newRatio));
  }

  onResize(newRatio);
}

function handlePointerUp(e: React.PointerEvent) {
  e.currentTarget.releasePointerCapture(e.pointerId);
  setDragging(false);
  onResizeEnd();
}
```

- Ratio clamping: `0.15` min, `0.85` max. Also enforced by `paneTree.resizePane`.
- Min pane size check: `120px` width, `80px` height.

function useMediaQuery(query: string): boolean {
  const [matches, setMatches] = useState(() => window.matchMedia(query).matches);
  useEffect(() => {
    const mql = window.matchMedia(query);
    const handler = (e: MediaQueryListEvent) => setMatches(e.matches);
    mql.addEventListener('change', handler);
    return () => mql.removeEventListener('change', handler);
  }, [query]);
  return matches;
}

## 성능 고려사항

- 최대 8개 `TerminalContainer` 동시 렌더링 (NFR-6106: <500MB)
- `SplitPane` 리렌더 범위 한정: `React.memo` on leaf components, keyed by `node.id`
- 드래그 중 IndexedDB 쓰기 금지 (`pointerup`에서만 저장)
- 모든 computed values에 `useMemo` 적용 (`paneCount`, `treeDepth`, `canSplit`, `allSessionIds`)
- `FitAddon.fit()` 호출은 `requestAnimationFrame`으로 배치 (NFR-6112)
- 리사이즈 중 16ms 이내 프레임 유지 (max 8 panes, flex reflow 비용 무시 가능)
