# Phase 4: 모바일 캐러셀

**SRS Reference**: FR-6301~FR-6307, ADR-605, ADR-608
**Depends On**: Phase 2 (usePaneManager, PaneRenderer)

---

## 목표

모바일(768px 이하)에서 동일한 `PaneLayout` 데이터를 횡 방향 캐러셀로 렌더링한다. 스와이프 전환, 도트 인디케이터, 롱프레스 컨텍스트 메뉴(바텀시트), 터치 이벤트 레이어링을 구현한다.

## 선행 조건

- Phase 2 완료 (usePaneManager, SplitPane, PaneRenderer)
- 기존: useResponsive (768px 브레이크포인트), AddTabModal 바텀시트 패턴

---

## 구현 체크리스트

### Task 4.1: PaneCarousel 컴포넌트

**파일**: `frontend/src/components/PaneSystem/PaneCarousel.tsx`

```tsx
interface PaneCarouselProps {
  leaves: PaneLeaf[];
  focusedIndex: number;
  onSwipe: (newIndex: number) => void;
  renderTerminal: (sessionId: string, paneId: string) => React.ReactNode;
  onLongPress: (paneId: string, position: { x: number; y: number }) => void;
}
```

**스와이프 엔진**:
1. 렌더링: `inline-flex` 컨테이너에 Pane들 배치, `transform: translateX(-${index * 100}%)`로 위치
2. 현재 + 이전/다음 3개 Pane 동시 마운트 (부드러운 전환)
3. 터치 처리:
   - `touchstart`: `startX`, `startY`, `startTime` 기록
   - `touchmove` (처음 50ms): `|deltaX| > |deltaY|` → 스와이프 모드 확정, `e.preventDefault()`
   - `touchmove` (스와이프 확정 후): `translateX` 실시간 업데이트
   - `touchend`: `|deltaX| > 50px` → snap to next/prev, 아니면 snap back
4. 바운스: 첫/마지막 Pane에서 역방향 스와이프 시 50px 이동 후 탄성 복귀
5. CSS transition: `300ms ease-out` (snapping 시), 드래그 중 `transition: none`

**터치 이벤트 레이어링 (ADR-608)**:
```
TouchLayer (투명, absolute, z-index 위, pointer-events: auto)
├─ touchstart → 좌표 기록, 50ms 타이머
├─ touchmove:
│   ├─ |X| > |Y| → 스와이프 모드, 이벤트 소비 (e.preventDefault)
│   └─ |Y| > |X| → this.style.pointerEvents = 'none' → xterm.js 위임
└─ touchend → 스와이프 완료 or 클린업, pointerEvents 복원
```

- xterm.js 터치 텍스트 선택: 500ms 롱프레스로 시작 → 50ms 판정과 충돌 없음
- `touch-action: pan-y` 설정으로 브라우저 기본 세로 스크롤 허용

```css
.pane-carousel {
  position: relative;
  overflow: hidden;
  touch-action: pan-y;
  width: 100%;
  height: 100%;
}
.pane-carousel-track {
  display: flex;
  height: 100%;
  will-change: transform;
}
.pane-carousel-track.snapping {
  transition: transform 300ms ease-out;
}
.pane-carousel-item {
  flex: 0 0 100%;
  width: 100%;
  height: 100%;
  overflow: hidden;
}
.pane-touch-layer {
  position: absolute;
  top: 0; left: 0; right: 0; bottom: 0;
  z-index: 10;
  pointer-events: auto;
}
```

### Task 4.3: PaneRenderer 모바일 분기 업데이트

```tsx
function PaneRenderer({ layout, isMobile, ...handlers }: PaneRendererProps) {
  if (isMobile) {
    const leaves = flattenPaneTree(layout.root);
    const focusedIndex = Math.max(0,
      leaves.findIndex(l => l.id === layout.focusedPaneId)
    );
    return (
      <div className="pane-mobile-container">
        <PaneIndicator
          total={leaves.length}
          current={focusedIndex}
          paneNames={leaves.map(l => `Terminal`)}
          onDotClick={(i) => handlers.onFocus(leaves[i].id)}
        />
        <PaneCarousel
          leaves={leaves}
          focusedIndex={focusedIndex}
          onSwipe={(i) => handlers.onFocus(leaves[i].id)}
          renderTerminal={handlers.renderTerminal}
          onLongPress={handlers.onLongPress}
        />
      </div>
    );
  }
  return <SplitPane node={layout.root} {...handlers} />;
}
```

### Task 4.4: 모바일 롱프레스 컨텍스트 메뉴 (FR-6205)

**트리거**: 터미널 500ms 터치 홀드 (이동 < 10px), `navigator.vibrate(50)` 진동 피드백

**바텀시트** (기존 AddTabModal 패턴):
```
┌────────────────────────┐
│ (어둠 배경)              │
├────────────────────────┤
│ 수평 분할 (위/아래)      │
│ 수직 분할 (좌/우)       │
│ ────────────           │
│ 줌 토글                 │
│ Pane 교환               │
│ ────────────           │
│ Pane 닫기               │
│ 다른 Pane 모두 닫기      │
│ ────────────           │
│ 프리셋 레이아웃 ▶       │ ← 탭 시 슬라이드
│ 레이아웃 저장            │
│ 레이아웃 불러오기 ▶     │ ← 탭 시 슬라이드
├────────────────────────┤
│         취소            │
└────────────────────────┘
```

서브메뉴: `transform: translateX(-100%)` + `transition: 300ms` 슬라이드, ← 뒤로가기 버튼

## 테스트

### 정상 케이스

| Given | When | Then |
|-------|------|------|
| 모바일(360px), 3 Pane | 화면 표시 | 캐러셀 1개 + 도트 3개 |
| 캐러셀 Pane 1/3 | 왼쪽 스와이프 (X>50px) | Pane 2로 전환 (300ms 애니메이션) |
| 캐러셀 | 도트 3 탭 | Pane 3으로 즉시 이동 |
| 모바일 터미널 | 500ms 롱프레스 | 바텀시트 메뉴 표시 |
| 바텀시트 "프리셋" | 탭 | 프리셋 서브메뉴 슬라이드 |

### 엣지 케이스

| Given | When | Then |
|-------|------|------|
| 첫 Pane | 오른쪽 스와이프 | 바운스, 이동 없음 |
| 터미널 스크롤백 | 세로 스와이프 | 캐러셀 무동작, 터미널 스크롤 |
| 3 Pane, 2번 표시 중 | Pane 닫기 | 1번으로 이동, 도트 2개 |
| 1개 Pane | 롱프레스 | "Pane 닫기" disabled |
| 데스크톱 4분할 | 768px 이하 리사이즈 | 캐러셀 + 도트 4개 |
