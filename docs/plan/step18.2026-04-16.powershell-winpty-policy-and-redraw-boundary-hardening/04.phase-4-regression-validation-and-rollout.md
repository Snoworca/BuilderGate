# Phase 4: Regression Validation And Rollout

## Objective

PowerShell 전용 producer 수정이 다른 셸, snapshot 구조, runtime reuse를 깨지 않는지 확인한다.

## Validation Matrix

### PowerShell

- plain PowerShell prompt
- invalid token rapid `A + Enter`
- `codex` launched inside PowerShell
- history recall
- tab completion menu
- multiline / wrapped line editing
- backspace / paste on long line
- resize while editing
- alternate-screen or child TUI 1종

### Other shells

- `bash`
- `wsl`
- `cmd`

### Existing runtime scenarios

- `tab -> grid`
- `grid -> tab`
- workspace switch
- reconnect / refresh

## Rollout Strategy

1. `windowsPowerShellBackend` field 도입, 기본값은 `inherit`
2. winpty capability probe와 fail-fast policy 도입
3. local / CI / clean-room 검증에서 explicit `winpty` cutover
4. mixed cohort 검증
5. telemetry / debug capture 확인
6. 검증 완료 후 example/default guidance 전환 여부 결정

### Mixed cohort policy

이 설정은 `new_sessions` 성격이다.

따라서 rollout 동안 아래 상태가 정상이다.

- legacy ConPTY PowerShell 세션
- new winpty PowerShell 세션

세션 continuity rule:

- session backend는 **session id 단위로 고정**된다
- reconnect / refresh / workspace restore는 backend를 바꾸지 않는다
- backend 전환은 새 세션 생성 또는 tab restart에서만 일어난다

운영 정책:

- rollout 중 기존 세션은 유지한다
- mixed cohort 종료가 필요하면 PowerShell 탭을 restart 또는 recreate 한다
- mass forced recycle은 기본 정책이 아니다

### Rollback policy

rollback trigger:

- explicit `winpty` probe failure
- PowerShell session creation failure spike
- PSReadLine compatibility regression
- child TUI regression

operator action:

1. incident response에서는 `windowsPowerShellBackend`를 명시적으로 `conpty`로 되돌림
2. 신규 PowerShell 세션부터 fallback policy 적용
3. 필요 시 affected workspace의 PowerShell 탭만 재시작
4. 최소 재검증:
   - plain PowerShell prompt
   - rapid `A + Enter`
   - `codex`
   - mixed cohort cleanup 확인

비고:

- `inherit`은 incident rollback 기본값이 아니다
- `inherit`은 사후 정리 단계에서만 사용 가능하며, 그 시점의 `pty.useConpty`가 검증된 값일 때만 허용한다

## Exit Criteria

- PowerShell rapid `A + Enter` 겹침이 재현되지 않음
- `replayEvents`가 0인 plain prompt 경로에서도 정상 적층
- non-PowerShell regressions 없음
- winpty unavailable 환경에서 정책이 결정적임
