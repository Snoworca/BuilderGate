## 4가지 인증 조합 요약

| COMBO | 조건 | 플로우 |
|-------|------|--------|
| COMBO-1 | `twoFactor.enabled = false` | 비밀번호 → JWT |
| COMBO-2 | `enabled=true`, `totp.enabled=false`, smtp 있음 | 비밀번호 → 이메일 OTP → JWT |
| COMBO-3 | `enabled=true`, `totp.enabled=true`, smtp 없음 | 비밀번호 → TOTP → JWT |
| COMBO-4 | `enabled=true`, `totp.enabled=true`, smtp 있음 | 비밀번호 → 이메일 OTP → TOTP → JWT |
