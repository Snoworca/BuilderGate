# Phase 3: Terminal Enhancement - Pinch-to-Zoom

## 목표 (Objective)

모바일 환경에서 터미널의 가독성을 향상시키기 위한 핀치-투-줌(Pinch-to-Zoom) 기능을 구현합니다. 사용자는 두 손가락 제스처로 터미널 폰트 크기를 8px~32px 범위에서 동적으로 조절할 수 있으며, 데스크톱 환경에서는 Ctrl+마우스 휠로 동일한 기능을 사용할 수 있습니다.

**주요 요구사항:**
- FR-1901: 2개 터치 포인트 감지 및 거리 계산
- FR-1902: 실시간 폰트 크기 조정 (8px~32px)
- FR-1903: 폰트 크기 로컬 스토리지 영속화

**기술 스택:**
- Pure Touch Events API (touchstart/touchmove/touchend)
- requestAnimationFrame for debouncing
- xterm.js `terminal.options.fontSize` API
- localStorage for persistence

## 아키텍처 참조 (Architecture Reference)

### ADR-005: Touch-Based Font Scaling Strategy

**결정 사항:**
- **Pure Touch Events API 사용** (no gesture libraries)
- **이유**:
  - 경량화: 추가 의존성 없음
  - 직접 제어: 정밀한 거리 계산 및 디버깅 용이
  - 브라우저 호환성: 모든 모던 브라우저 지원

**폰트 크기 범위:**
- 최소: 8px (가독성 하한선)
- 기본: 14px (현재 값 유지)
- 최대: 32px (모바일 화면 한계)
- 스텝: 1px (부드러운 조절)

**터치 이벤트 플로우:**
```typescript
touchstart (2 fingers detected)
  → capture initial distance (d₀)
  → save startFontSize
  → preventDefault() to block browser zoom

touchmove (2 fingers)
  → calculate current distance (d₁)
  → ratio = d₁ / d₀
  → newSize = clamp(round(startFontSize × ratio), 8, 32)
  → update terminal.options.fontSize via RAF
  → fitAddon.fit() to recalculate rows/cols

touchend
  → save final fontSize to localStorage('terminal_font_size')
  → show FontSizeToast for 1s
```

**거리 계산 공식:**
```typescript
distance = Math.hypot(
  touch1.clientX - touch2.clientX,
  touch1.clientY - touch2.clientY
)
```

**데스크톱 Fallback:**
```typescript
wheel event + ctrlKey
  → deltaY < 0: fontSize += 1
  → deltaY > 0: fontSize -= 1
  → clamp(8, 32)
```

### 3.1 usePinchZoom Hook 구현

**파일**: `frontend/src/hooks/usePinchZoom.ts` (NEW)

**요구사항 매핑**: FR-1901, FR-1902, FR-1903

**복잡도**: Medium

**상세 구현:**

```typescript
import { useEffect, useRef, useCallback } from 'react';

interface UsePinchZoomOptions {
  minSize: number;      // 8
  maxSize: number;      // 32
  defaultSize: number;  // 14
  storageKey: string;   // 'terminal_font_size'
  onFontSizeChange: (size: number) => void;
}

export function usePinchZoom({
  minSize,
  maxSize,
  defaultSize,
  storageKey,
  onFontSizeChange
}: UsePinchZoomOptions) {
  const startDistanceRef = useRef<number>(0);
  const startFontSizeRef = useRef<number>(defaultSize);
  const rafRef = useRef<number | null>(null);

  // Load initial font size from localStorage
  useEffect(() => {
    const saved = localStorage.getItem(storageKey);
    if (saved) {
      const size = parseInt(saved, 10);
      if (!isNaN(size) && size >= minSize && size <= maxSize) {
        startFontSizeRef.current = size;
        onFontSizeChange(size);
      }
    }
  }, [storageKey, minSize, maxSize, onFontSizeChange]);

  // Calculate distance between two touches
  const getDistance = useCallback((t1: Touch, t2: Touch): number => {
    return Math.hypot(
      t1.clientX - t2.clientX,
      t1.clientY - t2.clientY
    );
  }, []);

  // Handle touchstart
  const handleTouchStart = useCallback((e: TouchEvent) => {
    if (e.touches.length === 2) {
      e.preventDefault(); // Prevent browser zoom
      const distance = getDistance(e.touches[0], e.touches[1]);
      startDistanceRef.current = distance;
      startFontSizeRef.current = parseInt(
        localStorage.getItem(storageKey) || `${defaultSize}`,
        10
      );
    }
  }, [getDistance, storageKey, defaultSize]);

  // Handle touchmove with RAF debounce
  const handleTouchMove = useCallback((e: TouchEvent) => {
    if (e.touches.length === 2) {
      e.preventDefault();

      if (rafRef.current) {
        cancelAnimationFrame(rafRef.current);
      }

      rafRef.current = requestAnimationFrame(() => {
        const currentDistance = getDistance(e.touches[0], e.touches[1]);
        const ratio = currentDistance / startDistanceRef.current;
        const newSize = Math.round(startFontSizeRef.current * ratio);
        const clampedSize = Math.max(minSize, Math.min(maxSize, newSize));

        onFontSizeChange(clampedSize);
      });
    }
  }, [getDistance, minSize, maxSize, onFontSizeChange]);

  // Handle touchend - persist to localStorage
  const handleTouchEnd = useCallback((e: TouchEvent) => {
    if (rafRef.current) {
      cancelAnimationFrame(rafRef.current);
      rafRef.current = null;
    }

    if (e.touches.length < 2) {
      const currentSize = parseInt(
        localStorage.getItem(storageKey) || `${defaultSize}`,
        10
      );
      localStorage.setItem(storageKey, currentSize.toString());
    }
  }, [storageKey, defaultSize]);

  return {
    handleTouchStart,
    handleTouchMove,
    handleTouchEnd
  };
}
```

**체크리스트:**
- [ ] `getDistance` 함수: 유클리드 거리 계산
- [ ] `handleTouchStart`: 2 finger 감지, 초기 거리 저장, preventDefault
- [ ] `handleTouchMove`: RAF 디바운싱, 비율 계산, 크기 clamp
- [ ] `handleTouchEnd`: localStorage 저장
- [ ] 초기 로드: localStorage에서 폰트 크기 복원
- [ ] Cleanup: RAF 취소 로직

```typescript
import { useEffect, useState } from 'react';
import './FontSizeToast.css';

interface FontSizeToastProps {
  fontSize: number | null; // null when hidden
  duration?: number; // ms, default 1000
}

export function FontSizeToast({ fontSize, duration = 1000 }: FontSizeToastProps) {
  const [visible, setVisible] = useState(false);

  useEffect(() => {
    if (fontSize !== null) {
      setVisible(true);
      const timer = setTimeout(() => setVisible(false), duration);
      return () => clearTimeout(timer);
    }
  }, [fontSize, duration]);

  if (!visible || fontSize === null) return null;

  return (
    <div className="font-size-toast">
      {fontSize}px
    </div>
  );
}
```

### 3.3 TerminalView.tsx 수정 - Pinch Zoom 통합

**파일**: `frontend/src/components/Terminal/TerminalView.tsx` (MODIFY)

**요구사항 매핑**: FR-1901, FR-1902, FR-1903

**복잡도**: Medium

**수정 사항:**

```typescript
import { useEffect, useRef, useState, useCallback } from 'react';
import { Terminal } from '@xterm/xterm';
import { FitAddon } from '@xterm/addon-fit';
import { usePinchZoom } from '../../hooks/usePinchZoom';
import { FontSizeToast } from './FontSizeToast';
import { useResponsive } from '../../hooks/useResponsive'; // From Phase 1
import '@xterm/xterm/css/xterm.css';
import './TerminalView.css';

interface TerminalViewProps {
  // ... existing props
}

export function TerminalView(props: TerminalViewProps) {
  const terminalRef = useRef<Terminal | null>(null);
  const containerRef = useRef<HTMLDivElement | null>(null);
  const fitAddonRef = useRef<FitAddon | null>(null);

  const [currentFontSize, setCurrentFontSize] = useState<number | null>(null);
  const { isMobile } = useResponsive();

  // Font size change handler
  const handleFontSizeChange = useCallback((size: number) => {
    if (terminalRef.current) {
      terminalRef.current.options.fontSize = size;
      fitAddonRef.current?.fit(); // Recalculate rows/cols
      setCurrentFontSize(size); // Show toast

      // Hide toast after 1s
      setTimeout(() => setCurrentFontSize(null), 1000);
    }
  }, []);

  // Pinch zoom hook (mobile)
  const { handleTouchStart, handleTouchMove, handleTouchEnd } = usePinchZoom({
    minSize: 8,
    maxSize: 32,
    defaultSize: 14,
    storageKey: 'terminal_font_size',
    onFontSizeChange: handleFontSizeChange
  });

  // Desktop fallback: Ctrl+Wheel
  useEffect(() => {
    if (isMobile) return; // Only for desktop

    const handleWheel = (e: WheelEvent) => {
      if (e.ctrlKey) {
        e.preventDefault();
        const currentSize = terminalRef.current?.options.fontSize || 14;
        const delta = e.deltaY < 0 ? 1 : -1;
        const newSize = Math.max(8, Math.min(32, currentSize + delta));
        handleFontSizeChange(newSize);
        localStorage.setItem('terminal_font_size', newSize.toString());
      }
    };

    const container = containerRef.current;
    container?.addEventListener('wheel', handleWheel, { passive: false });
    return () => container?.removeEventListener('wheel', handleWheel);
  }, [isMobile, handleFontSizeChange]);

  // Attach touch events (mobile)
  useEffect(() => {
    if (!isMobile) return;

    const container = containerRef.current;
    if (!container) return;

    container.addEventListener('touchstart', handleTouchStart, { passive: false });
    container.addEventListener('touchmove', handleTouchMove, { passive: false });
    container.addEventListener('touchend', handleTouchEnd);

    return () => {
      container.removeEventListener('touchstart', handleTouchStart);
      container.removeEventListener('touchmove', handleTouchMove);
      container.removeEventListener('touchend', handleTouchEnd);
    };
  }, [isMobile, handleTouchStart, handleTouchMove, handleTouchEnd]);

  // ... existing xterm.js initialization code

  return (
    <div ref={containerRef} className="terminal-container">
      <div className="terminal" />
      {currentFontSize !== null && (
        <FontSizeToast fontSize={currentFontSize} />
      )}
    </div>
  );
}
```

**체크리스트:**
- [ ] `usePinchZoom` hook 통합
- [ ] `useResponsive` hook: isMobile 판단
- [ ] 터치 이벤트 리스너 등록 (모바일만)
- [ ] Ctrl+Wheel 이벤트 리스너 (데스크톱만)
- [ ] `handleFontSizeChange`: fontSize 업데이트 + fit() 호출
- [ ] `FontSizeToast` 렌더링
- [ ] localStorage 저장 (desktop wheel handler)
- [ ] Cleanup: 이벤트 리스너 제거
