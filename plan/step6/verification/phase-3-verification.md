## 1. Completion Checklist

### 1.1 ContextMenu 서브메뉴 지원 (FR-6204)
- [ ] `ContextMenu.tsx` 확장 완료
- [ ] `ContextMenuItem`에 `children?: ContextMenuItem[]` 필드 추가
- [ ] `children` 있는 항목에 `▶` 화살표 표시
- [ ] 마우스 호버 시 서브메뉴가 부모 오른쪽에 표시
- [ ] 화면 오른쪽 경계 초과 시 서브메뉴가 왼쪽에 표시 (overflow 방지)
- [ ] 서브메뉴 이탈 시 300ms 딜레이 후 닫힘

### 1.2 Pane 영역 컨텍스트 메뉴 (FR-6201)
- [ ] 터미널 영역 우클릭 시 Pane 컨텍스트 메뉴 표시
- [ ] "수평 분할 (위/아래)" — 단축키 표시 `Ctrl+B, "`, Pane 수 >= 8 또는 깊이 >= 4 시 비활성화
- [ ] "수직 분할 (좌/우)" — 단축키 표시 `Ctrl+B, %`, Pane 수 >= 8 또는 깊이 >= 4 시 비활성화
- [ ] 구분선
- [ ] "줌 토글" — 단축키 표시 `Ctrl+B, z`, 단일 Pane 시 비활성화
- [ ] "Pane 교환" — 단일 Pane 시 비활성화
- [ ] 구분선
- [ ] "출력 복사" — 선택 영역 또는 전체 텍스트 클립보드 복사
- [ ] 구분선
- [ ] "Pane 닫기" — 단축키 표시 `Ctrl+B, x`, 마지막 Pane 시 비활성화
- [ ] "다른 Pane 모두 닫기" — 단일 Pane 시 비활성화
- [ ] 줌 상태에서 "줌 해제"로 텍스트 변경, 분할/닫기 비활성화

### 1.3 경계선 컨텍스트 메뉴 (FR-6202)
- [ ] `PaneResizer` 우클릭 시 경계선 전용 메뉴 표시
- [ ] "균등 분할" — ratio를 0.5로 설정
- [ ] "방향 전환 (↔ ↕)" — horizontal ↔ vertical 전환
- [ ] 구분선
- [ ] "첫 번째 Pane 닫기" — 수직: "왼쪽", 수평: "위"
- [ ] "두 번째 Pane 닫기" — 수직: "오른쪽", 수평: "아래"

### 1.4 TabBar 메뉴 확장 (FR-6203)
- [ ] TabBar 탭 우클릭 메뉴에 항목 추가
- [ ] "레이아웃 저장" — 이름 입력 모달 → savedLayouts에 저장
- [ ] "레이아웃 불러오기" — 저장된 레이아웃 목록 모달
- [ ] "프리셋 레이아웃 ▶" — 서브메뉴: 단일, 좌우, 상하, 4분할, 1+2, 에이전트 모니터

### 1.5 레이아웃 저장/불러오기 모달 (FR-6502~FR-6504)
- [ ] 레이아웃 저장 모달: 이름 입력 (1~30자, 허용 문자 검증)
- [ ] 동일 이름 존재 시 덮어쓰기 확인 모달
- [ ] 레이아웃 불러오기 모달: 기본 프리셋 + 커스텀 목록, 이름/Pane수 표시
- [ ] 불러오기 전 확인 모달: "현재 레이아웃이 초기화됩니다. 계속하시겠습니까?"
- [ ] 커스텀 레이아웃 삭제 기능 (삭제 확인 모달)
- [ ] 기본 프리셋(`isBuiltIn=true`)에 삭제 옵션 미표시

### 1.6 빌드 및 통합
- [ ] `npm run build` 성공
- [ ] 기존 ContextMenu 기능 정상 동작 (탭 이름 변경, 탭 닫기 등)
