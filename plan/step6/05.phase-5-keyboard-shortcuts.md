## 구현 체크리스트

### Task 5.1: Prefix 모드 상태 머신 (TerminalView.tsx)

**수정 파일**: `frontend/src/components/Terminal/TerminalView.tsx`

기존 `customKeyEventHandler`에 Ctrl+B prefix 로직 추가:

```
상태 전이 다이어그램:
  NORMAL ──(Ctrl+B)──► PREFIX ──(키입력)──► 명령실행 ──► NORMAL
                        │                                    ▲
                        ├──(1500ms 타임아웃)─────────────────┘
                        ├──(Ctrl+B 재입력)──► \x02 전송 ──► NORMAL
                        └──(ESC)─────────────────────────► NORMAL
```

**구현 핵심**:

```typescript
// customKeyEventHandler 내부
if (e.type !== 'keydown') return true; // keyup 무시

if (!prefixMode) {
  // === NORMAL 상태 ===
  if (e.ctrlKey && e.key === 'b') {
    onEnterPrefixMode();  // usePaneManager.enterPrefixMode()
    return false;          // xterm에 전달 안함
  }
  // 기존 Ctrl+C, Ctrl+V 처리 유지
  return true;
} else {
  // === PREFIX 상태 ===
  onHandlePrefixKey(e.key, e.ctrlKey, e.shiftKey);
  return false; // PREFIX 모드에서는 항상 블록
}
```

**타임아웃 관리**:
- `enterPrefixMode()`: `setTimeout(exitPrefixMode, 1500)` 설정
- 키 입력 or ESC or 타임아웃 → `exitPrefixMode()` + `clearTimeout`
- `useEffect` 클린업에서 타이머 정리

**Props 추가**:
```tsx
interface TerminalViewProps {
  // 기존 props...
  prefixMode: boolean;
  onEnterPrefixMode: () => void;
  onHandlePrefixKey: (key: string, ctrl: boolean, shift: boolean) => void;
}
```

### Task 5.2: Prefix 키 매핑 (usePaneManager.handlePrefixKey)

`handlePrefixKey` 완성 구현:

```typescript
handlePrefixKey(key: string, ctrlKey: boolean, shiftKey: boolean): void {
  exitPrefixMode(); // 항상 prefix 해제

  // Ctrl+B, Ctrl+B → 실제 \x02 전송
  if (ctrlKey && key === 'b') {
    const leaf = findPane(layout.root, layout.focusedPaneId);
    if (leaf) sessionApi.sendInput(leaf.sessionId, '\x02');
    return;
  }

  switch (key) {
    // 분할
    case '%':           // Shift+5 → 수직 분할
      if (canSplit) splitPane(layout.focusedPaneId, 'vertical');
      break;
    case '"':           // Shift+' → 수평 분할
      if (canSplit) splitPane(layout.focusedPaneId, 'horizontal');
      break;

    // 포커스 이동
    case 'ArrowLeft':   moveFocus('left'); break;
    case 'ArrowRight':  moveFocus('right'); break;
    case 'ArrowUp':     moveFocus('up'); break;
    case 'ArrowDown':   moveFocus('down'); break;

    // Pane 조작
    case 'x':           // 닫기 (마지막 Pane이면 무시)
      if (paneCount > 1) closePane(layout.focusedPaneId);
      break;
    case 'z':           // 줌 토글
      toggleZoom();
      break;
    case 'q':           // 번호 오버레이 2초
      showPaneNumbers();
      break;
    case 'o':           // 다음 Pane 순환
      cycleFocus();
      break;

    // ESC → 이미 exitPrefixMode()했으므로 추가 동작 없음
    case 'Escape':
      break;

    // 알 수 없는 키
    default:
      showStatusMessage(`Unknown key: ${key}`, 1000);
      break;
  }
}
```

### Task 5.3: PaneNumberOverlay 컴포넌트

**파일**: `frontend/src/components/PaneSystem/PaneNumberOverlay.tsx`


**렌더링** (각 Pane 위에 오버레이):
```css
.pane-number-overlay {

  display: flex;

  font-size: 48px;

  color: white;
  text-shadow: 0 2px 4px rgba(0,0,0,0.5);
}
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}
```

**동작**:
1. `Ctrl+B, q` → `showPaneNumbers()` → `paneNumberOverlay = true`
2. 2초 타이머 시작 (`setTimeout(() => dismissOverlay(), 2000)`)
3. 표시 중 숫자키(0-7) 입력 → `selectPaneByNumber(num)` → 해당 Pane 포커스 + 오버레이 즉시 사라짐
4. 범위 밖 숫자 → 무시
5. 2초 후 자동 사라짐

**구현 위치**: 각 `PaneLeaf` wrapper 내부에 `paneNumberOverlay && <PaneNumberOverlay>` 조건 렌더링.

### Task 5.5: Ctrl+B 충돌 방지 (FR-6605)

- NORMAL 상태 `Ctrl+B` → PREFIX 진입, PTY에 전달하지 않음
- PREFIX 상태 `Ctrl+B` → PTY에 `\x02` 전송, PREFIX 해제
- 기존 터미널 앱(nano 등)에서 `Ctrl+B`(cursor back)는 **더블 Ctrl+B**로 사용

### Task 5.6: 에러 처리 (FR-6604)

- 인식되지 않는 키: `showStatusMessage('Unknown key: {key}', 1000)` → 1초 후 사라짐
- 불가능한 동작 (예: 단일 Pane에서 close): PREFIX 해제, 무동작 (에러 표시 안함)
- `showStatusMessage` 구현: StatusBar에 임시 메시지 표시, 타이머 후 클리어

### 정상 케이스

| Given | When | Then |
|-------|------|------|
| NORMAL 상태 | Ctrl+B | StatusBar `[PREFIX]` 표시 |
| PREFIX 상태 | % (Shift+5) | 수직 분할, PREFIX 해제 |
| PREFIX 상태 | " (Shift+') | 수평 분할, PREFIX 해제 |
| PREFIX 상태 | ArrowRight | 오른쪽 Pane 포커스 |
| PREFIX, 3 Pane | q | 각 Pane에 0, 1, 2 오버레이 2초 |
| 번호 오버레이 중 | '1' 입력 | Pane 1 포커스, 오버레이 사라짐 |

### 엣지 케이스

| Given | When | Then |
|-------|------|------|
| PREFIX 상태 | 1500ms 무입력 | PREFIX 자동 해제 |
| PREFIX 상태 | Ctrl+B 재입력 | PTY에 \x02 전송, PREFIX 해제 |
| PREFIX 상태 | 'a' (알 수 없는 키) | PREFIX 해제, "Unknown key: a" 1초 |
| PREFIX 상태 | ESC | PREFIX 해제, 무동작 |
| PREFIX, 단일 Pane | 'x' (close) | PREFIX 해제, 무동작 (닫을 수 없음) |
